## 三者差异

|                | cookie                 | localStorage | sessionStorage |
| -------------- | :--------------------- | :----------- | :------------- |
| 随着 http 发送 | 是                     | 否           | 否             |
| 最大数据量     | 4K                     | 5M           | 5M             |
| 失效时间       | 自定义或随会话断开失效 | 不清除不失效 | 随会话断开失效 |
| 跨域限制       | 是                     | 是           | 是             |
